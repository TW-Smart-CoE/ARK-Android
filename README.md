# Android-ARK

## System Requirements
* Ruby 2.6.5
* Bundler 2.1.4

## Setup(Mac)
- Install ruby & bundler
  ```sh
  brew install ruby
  bundle install
  ```
- Setup project
  go to project root dir, run this command in your terminal.
  ```sh
  ./scripts/setup.sh
  ```
